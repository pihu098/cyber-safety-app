
# 🛡️ Cyber Safety Advisor

Cyber Safety Advisor is a web-based cybersecurity platform that helps users detect online threats, verify websites and emails, and improve digital safety awareness.

## 🚀 Features
- Real-time website safety analysis  
- Email verification system  
- File scanning support  
- Cyber safety learning tools  
- Simple and interactive UI  

## 🛠️ Tech Stack
- Python (Flask)  
- HTML, CSS, JavaScript  
- MySQL Database  
- Deployed on Railway  

## ⚙️ Installation (Local Setup)

```bash
git clone https://github.com/your-username/cyber-safety-app.git
cd cyber-safety-app
pip install -r requirements.txt
python app.py
